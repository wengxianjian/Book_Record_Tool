# Book_Record_Tool

简单的看书记录工具

## 功能特点

- 📚 书籍管理：添加、编辑、删除书籍信息
- 📖 阅读状态：支持"想读"、"在读"、"已读"三种状态
- 📝 读书笔记：记录每本书的阅读笔记和感想
- 📅 年份查看：按年份筛选和查看已读书籍
- 📊 阅读统计：实时统计各状态书籍数量
- 👁️ 护眼主题：采用护眼配色方案，保护视力
- 🎨 字体调节：支持多种字体大小调节（8pt-24pt）

## 界面特点

- 窗口尺寸：1200 x 1200 像素
- 默认字体：18 pt
- 统一的字体大小，界面美观协调
- 清晰的标签页布局，操作便捷

## 使用方法

### 运行程序

直接运行 `Book_Record_Tool_v1.0.py` 文件：

```bash
python Book_Record_Tool_v1.0.py
```

### 打包为可执行文件

使用提供的 `build.bat` 脚本进行打包：

```bash
build.bat
```

打包完成后，可执行文件位于 `dist` 目录下。

## 数据存储

书籍数据默认保存在 `books_data.json` 文件中，采用 JSON 格式存储。

## 技术栈

- Python 3.x
- PyQt5
- JSON
